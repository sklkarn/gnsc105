The electronic structure of some of the Group IA elements are written as:
\begin{align*} H-1 \amp : 1s^1 \\ Li-3 \amp : 1s^2 2s^1\\ Na-11 \amp : 1s^2 2s^2 2p^6 3s^1\\ K-19 \amp : 1s^2 2s^2 2p^6 3s^2 3p^6 4s^1 \end{align*}
The electronic structure of some of the Group VIIIA elements are written as:
\begin{align*} He-2 \amp : 1s^2 \\ Ne-10 \amp : 1s^2 2s^2 2p^6\\ Ar-18 \amp : 1s^2 2s^2 2p^6 3s^2 3p^6 \end{align*}
The electronic structure of some other elements are written as:
\begin{align*} Be-4 \amp : 1s^2 2s^2\\ C-6 \amp : 1s^2 2s^2 2p^2\\ N-7 \amp : 1s^2 2s^2 2p^3\\ F-9 \amp : 1s^2 2s^2 2p^5\\ Mg-12 \amp : 1s^2 2s^2 2p^6 3s^2\\ Cl-17 \amp : 1s^2 2s^2 2p^6 3s^2 3p^5\\ Ca-20 \amp : 1s^2 2s^2 2p^6 3s^2 3p^6 4s^2 \end{align*}
From the given electronic structures we can find that Group IA elements contain only 1 electron in its outermost orbit which is the s -orbital (or, subshell). The Group IIA elements contain 3 electrons in its outermost orbit which is again the s-orbital and so on. The Group IIIA elements contain 3 electrons in its outermost orbit which is the p-orbital and so on. The Group VIIIA elements contain 2 electrons in helium atom and 8 electrons for rest of the atoms.
